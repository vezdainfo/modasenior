
Arquivos do site 'Moda Sénior' - Projeto Integrador
--------------------------------------------------

Estrutura:
- index.html
- produtos.html
- sobre.html
- contato.html
- acessibilidade.html
- /css/style.css
- /js/script.js
- /imagens/*    (logo.png, hero.jpg, prod1.jpg, prod2.jpg, prod3.jpg)

Observações:
1) As imagens incluídas são placeholders gerados automaticamente. Para usar imagens reais, recomendamos bancos de imagens gratuitos como Unsplash ou Pexels.
   - Sugestões de busca: "elderly fashion", "senior clothing", "elderly portrait", "senior couple smiling".
   - Baixe imagens em alta resolução e substitua os arquivos na pasta /imagens mantendo os mesmos nomes de arquivo se desejar manter o site funcionando sem alterar código.

2) A funcionalidade de leitura por voz usa a API SpeechSynthesis do navegador. Alguns navegadores ou sistemas operacionais podem exigir permissões ou não oferecer vozes em pt-BR.
   - Teste no Chrome ou Edge modernos para melhor compatibilidade.

3) Para hospedar localmente:
   - Abra o arquivo index.html no navegador (duplo clique) ou rode um servidor simples (por exemplo, 'python -m http.server' na pasta do site).

4) Para incluir imagens reais sem alterar nomes de arquivos, substitua os arquivos dentro da pasta /imagens por imagens com as mesmas dimensões ou proporções.

5) Se precisar, posso substituir os placeholders por imagens reais (Unsplash/Pexels) — caso você me autorize a buscar e incluir essas imagens. (Neste ambiente, não foram baixadas imagens externas automaticamente.)

Bom trabalho!
